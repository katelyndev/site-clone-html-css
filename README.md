💻 Site Clone - HTML e CSS

Projeto desenvolvido como desafio prático na Software Jr, com o objetivo de reproduzir fielmente o layout de uma landing page, utilizando apenas HTML e CSS.

🚀 Sobre o projeto

Este projeto consiste na criação de um clone de uma landing page, com foco em:

- Fidelidade ao layout original
- Organização do código
- Estruturação semântica em HTML
- Estilização responsiva com CSS

🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript 

📱 Responsividade

O site foi desenvolvido com foco em diferentes tamanhos de tela, adaptando-se para:

- Dispositivos móveis
- Desktops

🎯 Objetivo

Praticar conceitos fundamentais do desenvolvimento front-end, como:

- Estruturação de páginas web
- Estilização com CSS
- Responsividade
- Organização e boas práticas de código
